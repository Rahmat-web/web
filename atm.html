Project Gallry

ATM Management System (C# & SQL Server)

This project involved the development of a desktop-based ATM simulation using C# (Windows Forms) with Microsoft SQL Server as the robust backend. The primary objective was to create a secure and efficient system that accurately mimics real-world ATM operations. The system's core functionality includes secure PIN-based authentication with encrypted storage and configurable account lockouts, essential for protecting user data. It efficiently manages customer accounts, balances, and a detailed audit trail of transactions in the SQL Server database, utilizing parameterized queries and stored procedures to ensure data integrity and prevent security vulnerabilities like SQL injection. Key transaction modules, such as balance inquiries, cash withdrawals (incorporating overdraft protection), deposits, and transaction history viewing, are fully implemented, with each operation reliably updating the database. The user interface, built with Windows Forms, provides a realistic ATM screen simulation, complete with numeric keypads, prompts, and transaction logs displayed via DataGridView components. Strong data validation and exception handling are critical to the system's reliability, featuring thorough input checks and structured try-catch blocks for robust error management, which was verified through extensive testing and debugging for edge cases like insufficient funds or invalid PINs. Comprehensive documentation, including Entity-Relationship Diagrams (ERDs) and flowcharts , was created to clearly illustrate the system's architecture and data flow for thorough project presentation.
