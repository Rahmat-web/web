Project Gallry

Student & Faculty Relationship APP

I developed a fully functional Android application to facilitate seamless communication and engagement between students and faculty members. The app included features such as birthday notifications, event sharing, automated replies, and expressive emoji/sticker messaging to enhance user interaction. To ensure smooth management, I created admin panels for handling users, events, and system data, while integrating a secure and robust back end using Android Studio, Retrofit, .NET Web API, and SQL Server.

Alongside development, I conducted a detailed evaluation of communication breakdowns within the project team and across stakeholders to identify inefficiencies and improve collaboration. This was supported by comprehensive documentation, including requirement specifications, meeting notes, and implementation plans to guide execution. I designed ERDs, process flowcharts, and system architecture diagrams to visualize workflows and data structures, while also creating intuitive UI/UX mockups and interactive prototypes to elevate usability and consistency. Additionally, I prepared analytical reports with proper citations to reinforce project decisions and enhance presentation clarity.

Tools & Technologies

  • Figma: Wireframes, UI/UX design & prototypes
  • Adobe Photoshop & Canva: Graphic design & visual enhancements
  • Android Studio: Android app development
  • Retrofit: API integration & network communication
  • .NET Web API: Backend development & data processing
  • SQL Server: Database management
  • Project Duration: 2023–2024
  • Client: Final Year Project
